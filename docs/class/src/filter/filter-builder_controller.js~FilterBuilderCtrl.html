    FilterBuilderCtrl

    The FilterBuilderCtrl class provides a controller for the <filter-builder> directive; it contains logic for extracting and combining the ng-model of any number of arbitrary transcluded inputs inside it.

    The values of its transcluded inputs are converted to a unified query filter object (MongoDB/Mongoose-style in this case).

    Unlike the <filter-controls> component, this directive does not make any server calls or deal otherwise with with the actual map data – it is purely responsible for providing the filter to make the server calls with.
