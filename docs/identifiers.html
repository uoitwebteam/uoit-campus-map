    FilterBuilderCtrl

    The FilterBuilderCtrl class provides a controller for the <filter-builder> directive; it contains logic for extracting and combining the ng-model of any number of arbitrary transcluded inputs inside it.
    public

    MapControlsCtrl

    The MapControlsCtrl provides an interface between the map component and the map control filter dropdowns.
    public

    MapCtrl

    The MapCtrl is the lead orchestrator of the component: it wraps the NgMap directive's methods and provides its own for interfacing with the map's controls, generating dialogs, and re-rendering the map elements.
    public

    MapDetailCtrl

    The MapDetailCtrl is a small controller with few responsibilties – it is only instantiated by ng-material's $mdPanel service as a means to provide data from a clicked feature to a dialog window that will display the data; it also contains transition methods for viewing tour items (e.g. from the dialog's "Take a tour" button).
